Как добавить оперативной памяти или ресурсов процессора виртуальной машине?**  
Редактиреем файл `Vargant`:
```
Vagrant.configure("2") do |config|
    config.vm.box = "bento/ubuntu-20.04"
    config.vm.provider "virtualbox" do |v|
	    v.memory = 2048
	    v.cpus = 2
    end
end
```
